JavaScript String Methods

JavaScript String Length

The JavaScript length property is used to determine the length of a string, which represents the number of characters in the string. It returns an integer value representing the length of the string.

JavaScript String slice()

The JavaScript slice() method is used to extract a portion of a string and return it as a new string, without modifying the original string. It takes two optional parameters: start and end. The start parameter specifies the index at which the extraction should begin (inclusive), and the end parameter specifies the index at which the extraction should end (exclusive). If the end parameter is omitted, the extraction will include all characters from the start index until the end of the string.

Replacing String Content

In JavaScript, you can replace specific content within a string using the replace() method. The replace() method searches for a specified value or regular expression pattern in a string, and replaces it with a new value. It returns a new string with the replaced content.

JavaScript String toUpperCase()

The toUpperCase() method in JavaScript is used to convert a string to uppercase letters. It returns a new string where all the characters in the original string are converted to uppercase.

JavaScript String toLowerCase()

The toLowerCase() method in JavaScript is used to convert a string to lowercase letters. It returns a new string where all the characters in the original string are converted to lowercase.

JavaScript String concat()

The concat() method in JavaScript is used to concatenate, or join, two or more strings together. It returns a new string that is the result of combining the original string with the provided strings.

JavaScript String trim()

The trim() method in JavaScript is used to remove leading and trailing whitespace from a string. It returns a new string with the whitespace trimmed.

const str = " Hello, World! ";

const trimmedStr = str.trim();

console.log(trimmedStr); // Output: "Hello, World!"

JavaScript String Padding

String padding in JavaScript refers to adding characters to the beginning or end of a string to achieve a desired length. JavaScript provides two methods for string padding: padStart() and padEnd(). The padStart() method is used to pad the beginning of a string, while the padEnd() method is used to pad the end of a string. Both methods take two parameters: the target length and the padding character(s).

JavaScript String charAt()

JavaScript String charAt() method is used to retrieve the character at a specified index within a string. It takes the index as a parameter and returns the character at that position.

Here's an example:

const str = "Hello";

const character = str.charAt(1);

console.log(character); // Output: "e"

JavaScript String split()

JavaScript String split() method is used to split a string into an array of substrings based on a specified separator. The original string is divided at each occurrence of the separator, and the substrings are stored as elements in the resulting array.

Here's an example:

const str = "Hello,World,JavaScript";

const arr = str.split(",");

console.log(arr); // Output: ["Hello", "World", "JavaScript"]
